Input

설명이 있는 기본 인풋 스타일

<div class="input-type">
    <label for="label-name">내용 설명</label>
    <input type="text" id="label-name" name="label-name" />
</div>

설명이 없는 기본 인풋 스타일

<div class="input-type">
    <input type="text" aria-label="내용 설명" />
</div>

설명이 없는 너비 100% 인풋 스타일

<div class="input-type full">
    <input type="text" aria-label="내용 설명" />
</div>

input-type 클래스의 형재요소 클래스로 full 을 추가해준다.
ex) <div class="input-type full">

Placeholder가 있는 인풋 스타일

<div class="input-type">
    <input type="text" placeholder="Placeholder 내용" aria-label="내용 설명" />
</div>

여러개 버튼 구성 시

기본 버튼 기본 버튼 기본 버튼 
<div class="btn-set-area">
    <form class="btn-size btn-blue">
        <label class="btn-icon">버튼</label>
        <input type="button" name="search-keyword">
    </form>
    <a href="" class="btn-size btn-defult" role="button">버튼</a>
    <button class="btn-size btn-blue" type="button">버튼</button>
</div>
"btn-set-area" class 내부에 버튼을 추가 하고 각 버튼의 특성에 따라 엘리먼트에 설정하여 class를 적용 하시면 됩니다.

아이콘 버튼 스타일

<form class="btn-size btn-blue">
    <label class="btn-icon">버튼</label>
    <input type="button" name="search-keyword">
</form>
"btn-size" class로 사이즈 설정, "btn-blue" class로 색상 설정, 아이콘이 있는 버튼의 경우 "btn-icon" class로 아이콘 설정

기본 버튼 스타일

버튼 
<a href="" class="btn-size btn-defult" role="button">버튼</a>
기본 디폴트 버튼 "btn-defult" class로 색상 설정

활성화 버튼 스타일

<button class="btn-size btn-blue" type="button">버튼</button>
기본 활성화 버튼 "btn-blue" class로 색상 설정

Popup Button

팝업버튼 클릭 
<a class="btn-popup-name" role="button" href>팝업 버튼 클릭</a>

팝업버튼의 class="btn-popup-name" 클래스명 클릭 시 해당 팝업의 data-id="popup-name" 와 연결되어 팝업이 띄워지도록 설정 되었습니다.
클래스의 popup-name 은 파일 네이밍과 동일하게 규칙적으로 사용합니다.

Popup Base Style

<div class="popup" data-id="popup-name">
